rand([x[,y]]) => z

Arguments

x lower/upper bound
y upper bound

Returns

z random number

Descriptions

Generates a new random number within the specified range. If no aruments are provided then a random value between 0 and the maximum random value will be generated. If just x is provided then a random number will be generated between 0 and x. If x,y is provided then a random number will be generated between x and y.
